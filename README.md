
🎵 Automação Spotify + YouTube - play-apotecaria

Este projeto consiste em uma automação desenvolvida em Python que realiza a abertura automática do navegador, acessa as plataformas Spotify e YouTube e executa a música e o vídeo da abertura do anime Diários de uma Apotecária.

📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar automação de tarefas utilizando Python.

O script simula interações do usuário no sistema, como abertura do navegador, digitação de URLs, uso de atalhos de teclado e execução automática de mídia em plataformas de streaming.

🛠 Tecnologias utilizadas

Python

PyAutoGUI

Time

Automação de Interface Gráfica (GUI Automation)

⚙️ Como funciona

O script abre o navegador

Acessa o Spotify

Reproduz a música da abertura

Acessa o YouTube

Dá play no vídeo da abertura

▶️ Como executar
python nome_do_arquivo.py
🎯 Objetivo do projeto

Praticar automação com Python, simulando interações humanas no sistema operacional para executar tarefas repetitivas de forma automática.
